# preterm-birth-prediction-ml
# 🏥 Preterm Birth Prediction using Machine Learning

This project focuses on the development of a clinical decision-support system for predicting preterm birth using machine learning techniques.

## 📌 Overview

Preterm birth (before the 37th week of gestation) is a major cause of neonatal complications.
This project uses machine learning models to predict the risk and timing of preterm birth based on clinical data.

## ⚠️ Dataset

The dataset used in this project contains sensitive clinical data from a hospital and is not publicly available due to privacy and ethical restrictions.

## ⚙️ Methods

* Data preprocessing & feature engineering
* Handling class imbalance (SMOTE)
* Bias mitigation techniques
* 10-fold cross-validation

### Models used:

* Random Forest
* Support Vector Machine (SVM)
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree
* Gradient Boosting
* AdaBoost
* XGBoost
* Naive Bayes
* Bagging
* Optimized Voting

## 📈 Results

* High-performing classification models with strong F1-score and ROC-AUC
* Random Forest Regressor achieved strong performance for gestation prediction

## 🎯 Goal

To support early clinical decision-making and improve outcomes for preterm birth.
